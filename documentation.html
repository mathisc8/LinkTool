Link Dashboard Documentation

Documentation created by me – an overview of all features and functionality.

1. Main Navigation

The interface is designed with simplicity and efficiency in mind. The layout includes a sidebar and a header:

Sidebar

  • Main Menu: Quickly switch between the “All Links” and “Favorites” views.
  • Folders: Organize your links by categories. Each folder displays the number of links it contains.
  • Theme Toggle: Switch between light and dark modes using the dedicated button.

Header

  • New Link: Click this button to open the form for adding a new link.
  • Toggle Dates: Show or hide the date when each link was added.
  • Notes: Open the notes panel to manage your personal notes and notebooks.
  • Profile & Logout: Your email is displayed (fetched via Supabase authentication), and you can open a profile dropdown to log out.

2. Link Management

Adding a New Link

  1. Open the Form: Click the “New Link” button in the header.
  2. Enter the Information:
    • Title: Optional. If left blank, the system will extract a title from the URL.
    • URL: Type the website address (e.g., example.com). The system automatically adds the https:// prefix when needed and normalizes the URL (removing unnecessary parameters and “www.”).
    • Folder: Choose a folder to assign the link or leave it unassigned.
  3. Submit or Cancel: Click “Add” to save the link or “Cancel” to dismiss the form.

Editing or Deleting a Link

  • Edit: Click the pencil icon to modify a link’s details.
  • Delete: Click the trash icon to remove a link. A confirmation prompt may appear.

Marking as Favorite

Click the star icon to add or remove a link from your favorites. Favorite links are displayed in a dedicated view.

Additional Actions

  • Preview: Opens a modal with a live preview of the website. Preloading can be enabled for faster access when you have many links.
  • QR Code: Generates and allows you to download a QR code for sharing the link.
  • Drag & Drop: Easily reorder your links by dragging and dropping them into the desired order.
  • Undo: Use the keyboard shortcut Ctrl+Z (or Cmd+Z on Mac) to undo your last action—whether it was adding, editing, or deleting a link.

3. Folder Management

Creating a Folder

  1. Click the “New Folder” button in the sidebar or above the folder list.
  2. Enter a name for your folder (for example, “Work” or “Personal”).
  3. Confirm to create the folder. Once created, you can assign links to it.

Renaming or Deleting a Folder

  • Rename: Right-click on a folder and select “Rename” to change its name.
  • Delete: Choose “Delete” from the context menu. All links inside will automatically be moved back to “All Links.”

Reordering Folders

Reorder your folders by dragging and dropping them into your preferred sequence.

4. Notes Management

Using the Notes Panel

  • Click the “Notes” button in the header to open the notes panel.
  • Create Notebooks to group your notes by topic.
  • Within each notebook, click “New Note” to add a note.
  • Edit the note’s title and content using the built-in editor (which supports formatting like bold, italic, lists, etc.).

Organizing and Rearranging Notes

Drag and drop notes to reorder them within a notebook or to move them between notebooks. Changes are saved live and synchronized with Supabase.

5. Other Features

Preview and Preloading

  • Live Preview: Opens a modal window that displays a live preview of the linked website.
  • Preloading: Enable preloading to fetch and cache preview data, which is especially useful when managing many links.

Display Options

  • Theme: Toggle between light and dark modes using the dedicated button.
  • Dates: Choose to display or hide the date a link was added.

Backup and Synchronization

All changes—including adding, editing, or deleting links, folders, and notes—are saved and synchronized via Supabase in real time.

Keyboard Shortcuts

Use Ctrl+Z (or Cmd+Z on Mac) to undo your last action.

6. Tips

  • Keep Organized: Regularly sort your links into folders to keep your dashboard tidy.
  • Use Favorites: Mark important links as favorites for quick access.
  • Preview First: Use the preview function to quickly check a link’s content without leaving the dashboard.
  • Notes Are Handy: Utilize the notes panel to save useful information and ideas.
  • Drag & Drop: Easily reorder links, folders, and notes by dragging and dropping them.
  • Automatic URL Processing: URLs are automatically normalized (adding protocols, removing “www.”, cleaning up query parameters) to prevent duplicate entries.
